# 📊 Sales Data Analysis using SQL & Power BI

## 📌 Project Overview

This project focuses on analyzing sales data to uncover meaningful business insights and support data-driven decision-making. The analysis was performed using SQL for data querying and Power BI for visualization.

---

## 🎯 Business Objective

The main goal of this project is to:

* Evaluate overall sales performance
* Identify profit-driving and loss-making areas
* Analyze customer purchasing behavior
* Support business strategy with actionable insights

---

## 🛠️ Tools & Technologies Used

* **SQL** – Data extraction and analysis
* **Power BI** – Dashboard creation and visualization
* **Excel** – Data cleaning and preprocessing

## 📊 Key Insights

* Technology category generated the highest revenue
* Furniture category showed consistent losses in some regions
* High discounts negatively impacted profit margins
* Repeat customers contributed significantly to overall sales

* Sales & Profit Overview

* Region-wise Sales Distribution
* Customer Segmentation
* Interactive Filters for dynamic analysis

---

## ⚠️ Business Problems Identified

* Excessive discount strategy reducing profitability
* Underperforming product categories
* Regional imbalance in sales performance
